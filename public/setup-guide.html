Enable write access to Google Sheets in 5 minutes

‚ùå Current Error: "Failed to add member to sheet. The API key may need write permissions."
üí° Why this happens: Google Sheets API with just an API key can only read data, not write. You need to set up a Google Apps Script Web App to enable adding members.

1 Open Google Apps Script

Go to your Google Sheet and open the script editor:

  • Open: Your Google Sheet
  • Click Extensions ‚Üí Apps Script

2 Copy & Paste the Script

Delete all existing code and paste this:

function doPost(e) { try { const data = JSON.parse(e.postData.contents); if (data.action === 'addMember') { const ss = SpreadsheetApp.getActiveSpreadsheet(); const sheet = ss.getSheets()[0]; // Get current timestamp const timestamp = new Date(); const rowData = [ timestamp, // Timestamp (Column A) data.data.name || '', // Name (Column B) data.data.village || '', // Village (Column C) data.data.currentAddress || '', // Current Address (Column D) data.data.mobile || '', // Mobile (Column E) data.data.member2Name || '', data.data.member2Age || '', data.data.member2Business || '', data.data.member3Name || '', data.data.member3Age || '', data.data.member3Business || '', data.data.member4Name || '', data.data.member4Age || '', data.data.member4Business || '', data.data.member5Name || '', data.data.member5Age || '', data.data.member5Business || '', data.data.member6Name || '', data.data.member6Age || '', data.data.member6Business || '', data.data.remark || '' ]; sheet.appendRow(rowData); return ContentService.createTextOutput(JSON.stringify({ status: 'success', message: 'Member added successfully' })).setMimeType(ContentService.MimeType.JSON); } } catch (error) { return ContentService.createTextOutput(JSON.stringify({ status: 'error', message: error.toString() })).setMimeType(ContentService.MimeType.JSON); } }

Important: Click the Save button (üíæ) or press Ctrl+S

3 Deploy as Web App

  1. Click Deploy ‚Üí New deployment
  2. Click the gear icon ‚öôÔ∏è ‚Üí Select Web app
  3. Configure settings:
    • Execute as: Me
    • Who has access: Anyone
  4. Click Deploy
  5. Click Authorize access ‚Üí Choose your Google account
  6. Click Advanced ‚Üí Go to [project name] (unsafe) ‚Üí Allow
  7. COPY THE WEB APP URL (looks like: https://script.google.com/macros/s/AKfycb.../exec)

4 Update .env File

Open your .env file and add this line:

VITE_GOOGLE_SCRIPT_WEB_APP_URL=https://script.google.com/macros/s/YOUR_URL_HERE/exec

Replace YOUR_URL_HERE with the URL you copied in step 3.

5 Restart Development Server

In your terminal:

# Press Ctrl+C to stop the server npm run dev
